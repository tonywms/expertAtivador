🚀 WMS Expert Activator API

Sistema de geração de chaves de ativação para WMS Expert

📌 Endpoint Principal

GET /api/gerar

📝 Parâmetros:

  • hwid - Hardware ID (ex: ABC-123DEF)
  • validade - Data de expiração (formato: DD/MM/AAAA)

🔧 Exemplo de uso:

https://expert-ativador.vercel.app/api/gerar?hwid=ABC-123DEF&validade=25/12/2025

📤 Resposta:

{ "sucesso": true, "chave_gerada": "xyz-abc-FICDICFDFJ-EFFEGHIHEFGGFE", "hwid_origem": "ABC-123DEF", "expira_em": "25/12/2025" }

💡 Exemplo no Delphi:

// Usando TIdHTTP
url := 'https://expert-ativador.vercel.app/api/gerar?hwid=' +
    EditHWID.Text + '&validade=' + EditValidade.Text;
response := IdHTTP1.Get(url);
